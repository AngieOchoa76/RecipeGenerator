
# Recipe Manager

A command-line application that allows users to create, view, and analyze recipes.
This project demonstrates list manipulation, loops, input validation, and modular program design in Python.

---

## Features
- Add a new recipe with validation  
- View all recipes  
- Display recipe statistics  
- Search recipes under a maximum cook time  
- Uses parallel lists to store recipe data
